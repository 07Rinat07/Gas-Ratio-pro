# Gas Ratio Interpreter v0.3

Локальное инженерное приложение для импорта газовых данных, сопоставления колонок,
расчета газовых коэффициентов, построения Pixler/ternary палеток и предварительной
интерпретации интервалов.

## Стек

- Python 3.11+
- Streamlit
- pandas, numpy
- plotly
- openpyxl
- pytest

## Установка

```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app/streamlit_app.py
```

## Поддерживаемые форматы

- CSV
- XLSX
- XLSM

Для Excel читаются все листы. Для CSV автоматически определяется разделитель,
поддерживаются кодировки `utf-8-sig`, `utf-8`, `cp1251`, `latin1`.

## Порядок колонок

Фиксированный порядок колонок не нужен. Приложение сначала читает сырые данные,
ищет строку заголовков среди первых 50 строк, затем выполняет mapping колонок.
Пользователь может вручную выбрать строку заголовков и исправить сопоставление.

Поддерживаемые стандартные поля:

```text
well, depth, depth_from, depth_to,
c1, c2, c3, ic4, nc4, ic5, nc5,
co2, h2s, rop, lithology
```

Если компонент C1-C5 отсутствует, он принимается равным 0 с предупреждением.
Если глубина отсутствует, используется индекс строки как техническая глубина
с предупреждением.

## Формулы

Основные расчеты:

```text
ΣC4 = iC4 + nC4
ΣC5 = iC5 + nC5
ΣC  = C1 + C2 + C3 + ΣC4 + ΣC5

Wh   = (C2 + C3 + ΣC4 + ΣC5) * 100 / ΣC
Bh   = (C1 + C2) / (C3 + ΣC4 + ΣC5)
BAR2 = C1 / C2
```

Pixler:

```text
C1/C2
C1/C3
C1/ΣC4
C1/ΣC5
```

Ternary:

```text
C2/ΣC
C3/ΣC
nC4/ΣC
```

Формула Ch требует подтверждения по корпоративной методике. Подробности см.
в `docs/formulas.md`.

## Ограничения v0.3

- Интерпретация является предварительной инженерной подсказкой.
- Pixler-зоны вынесены в конфиг и требуют замены на подтвержденные корпоративные линии.
- Ternary построен на базовом Plotly ternary и позже может быть заменен кастомной палеткой.
