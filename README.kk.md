## Негізгі мүмкіндіктер

- LAS 1.x/2.x/3.x, DLIS/LIS79 және SEG-Y метадеректерін алдын ала қарау;
- Data Platform, өзгермейтін артефактілер, Dataset Manifest, lineage және provenance;
- LAS QC, PDF/DOCX есептері, корреляция және интерпретация;
- орыс, қазақ және ағылшын тілдеріндегі интерфейс пен нұсқаулықтар.

## Орнату және іске қосу

Python 3.10+ қажет.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
.\run_app.ps1
```
