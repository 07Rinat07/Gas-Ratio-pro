# Correlation Workspace Redesign Specification v4.0

## Статус

Обязательный документ Roadmap v4.0.

## Проблема

Текущая вкладка корреляции показывает скважины и LAS-файлы разрозненно. Пользователю непонятно:

- какие скважины участвуют в корреляции;
- какие LAS-файлы назначены скважинам;
- какие кривые используются;
- где маркеры пластов;
- где контакты;
- по каким правилам построена корреляция.

## Цель

Создать профессиональное рабочее пространство корреляции, в котором пользователь видит полный workflow: выбор скважин, кривых, маркеров, интервалов, tie-lines и результата.

## Основные компоненты

### 1. Well Selection Panel

- список скважин проекта;
- назначенный LAS для каждой скважины;
- статус данных;
- быстрый preview кривых;
- предупреждение, если LAS отсутствует.

### 2. Correlation Setup Panel

- выбор базовой скважины;
- выбор коррелируемых скважин;
- выбор кривых корреляции;
- выбор интервала глубин;
- выбор режима шкалы.

### 3. Marker Manager

- formation tops;
- stratigraphic markers;
- OWC/GOC/GWC/FWL;
- pay zones;
- user markers;
- импорт/экспорт маркеров.

### 4. Tie Line Manager

- ручное создание линий корреляции;
- редактирование;
- удаление;
- проверка пересечений;
- отображение ошибок.

### 5. Correlation Display

- синхронизированные tracks;
- общая глубинная шкала;
- ручной ввод depth from/depth to;
- scale controls;
- zoom/pan;
- crosshair;
- подписи скважин;
- легенда кривых и маркеров.

### 6. Export

- PDF;
- PNG;
- SVG;
- TIFF;
- report package.

## Критерии приемки

- пользователь может выбрать минимум две скважины и явно увидеть, что именно коррелируется;
- каждая скважина имеет понятную подпись и источник LAS;
- маркеры и tie-lines видимы;
- есть ручной диапазон глубины;
- есть экспорт корреляционной панели;
- пустое состояние не выглядит как ошибка.
