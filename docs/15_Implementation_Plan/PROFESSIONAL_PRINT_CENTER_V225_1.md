# Professional Print Center v225.1

## Цель

Полностью упростить печать инженерных отчётов и исключить повторные нажатия при формировании файла.

## Реализовано

- Главный экран печати сокращён до языка, формата, диапазона и параметров страницы.
- Профиль, режим, шаблон, состав разделов и служебные параметры перенесены в закрытый блок дополнительных настроек.
- Проверка состава отчёта перенесена в закрытый диагностический блок.
- Во время активного экспорта основная кнопка блокируется и показывает состояние формирования.
- Для одного проекта одновременно допускается только одна активная задача экспорта.
- Увеличены печатные размеры заголовков треков, статистики, шкал глубины и осей.
- Увеличена полезная высота печатного инженерного планшета.

## Следующие задачи

- Встроенный постраничный предпросмотр PDF.
- Раздельные физические профили A4/A3 без уменьшения текста ниже минимального размера.
- Полная унификация SVG/PDF/DOCX через один page-aware renderer.
