# Gas Ratio Pro v225.11 Stable

Stage 5.2 добавляет project-scoped импорт операторских калибровочных ZIP-пакетов, data-rights gate, immutable source/rights fingerprints, сравнение версий и versioned project authorization packages. Production formulas не изменялись. Private operator data не включаются в релизный архив.

## Основные изменения

- безопасный трёхфайловый ZIP-контракт без каталогов и path traversal;
- проверка project scope, owner, legal basis, data classification, processing, derivative analysis, final-report use, redistribution и expiration;
- SHA-256 и размер каждого payload, package fingerprint и method-registry fingerprint;
- неизменяемое project-scoped хранение исходного ZIP и JSON-контрактов;
- сравнение baseline/operator и operator/operator по 10 методам;
- versioned project authorization package перед финальным экспортом;
- Export History schema v5;
- трёхъязычная панель Professional Print Center;
- Foundation Dual Water остаётся `blocked_final_report`.
