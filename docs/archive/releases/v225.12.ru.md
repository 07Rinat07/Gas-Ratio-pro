# Gas Ratio Pro v225.12 Stable

Stage 5.3 добавляет detached Ed25519 signatures, application trust registry, reviewer approval, revocation, expiry monitoring, signed package lineage и контролируемое продвижение `development → validation → production`. Исходный операторский ZIP и production formulas не изменяются.

## Основные изменения

- detached signature contract для точного package fingerprint;
- public-key trust registry с project/environment scope и validity periods;
- immutable review decisions и revocation records;
- technical/data-governance approval policy;
- expiry monitor для data rights, signature и key;
- signed lineage и защита от cycle/self-reference/conflicting parent;
- promotion records и проверка целостности environment state;
- production trust gate до activation, PresentationModel и renderer;
- trust evidence в project authorization, ExportArtifact и Export History v6;
- трёхъязычный trust workflow в Professional Print Center;
- private keys исключены из приложения и релизного архива;
- production formulas не изменялись; Foundation Dual Water остаётся `blocked_final_report`.
