# Visualization Curve Quality Engine v84

## Назначение

Слой качества кривых выполняет подготовку данных до renderer. Он разбивает кривые на независимые сегменты, не допускает соединения через пропуски и отсекает точки за пределами текущего depth viewport.

## Реализовано

- разрыв сегмента на `None`, `NaN`, бесконечных и недопустимых логарифмических значениях;
- разрыв при обратном ходе глубины и аномально большом depth gap;
- clipping по диапазону глубины;
- нормализация linear и logarithmic axes;
- отдельные `polyline` primitives для каждого валидного сегмента;
- quality metadata и диагностические сообщения в Render Model.

## Архитектурная граница

Renderer получает только готовые pixel coordinates. Он не анализирует исходные значения кривой, пропуски, шкалу и глубинный диапазон.
