# LAS Viewer Bookmark Trash v155

Version 155 adds recoverable deletion for recent LAS session bookmarks.

## Capabilities

- Removed bookmarks are moved to persistent trash instead of being destroyed.
- Bookmark label, folder, and position are preserved.
- A bookmark can be restored only while the corresponding recent LAS session exists.
- One trash item or the entire trash can be permanently purged.
- The contract remains renderer-neutral and is stored with recent-session preferences.
