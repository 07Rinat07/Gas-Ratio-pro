# GAS RATIO PRO v225.4

Релиз подключает единый физический page-aware package непосредственно к видимому Professional Print Center. Пользователь до экспорта видит точный профиль и все страницы. DOCX и HTML получают тот же канонический multi-page preview без повторного расчёта layout.

Сборка сохраняет строгий запрет fallback на первую страницу и синхронизирует подписи и диагностику на русском, казахском и английском языках.
