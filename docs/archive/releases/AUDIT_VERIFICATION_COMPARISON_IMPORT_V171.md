# Audit verification comparison import v171

Добавлены чтение и детерминированное объединение экспортированных comparison reports.

Основные свойства:

- проверка SHA-256 и структуры каждого входного файла до объединения;
- отсутствие изменения состояния repository при ошибке;
- дедупликация событий;
- выявление конфликтов, когда одно событие одновременно добавлено и удалено;
- renderer-neutral итоговый контракт.
