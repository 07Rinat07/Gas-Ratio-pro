# GAS RATIO PRO v225.3

This release combines physical page chrome, page numbering, and a repeated legend with one page-aware export package. SVG, PDF, and PNG share one geometry source; Print Center receives the exact profile and page count. LAS Viewer no longer coordinates SVG and PDF independently.

The release also restores true executor-backed background dispatch: the UI can observe intermediate progress and cooperative cancellation is processed before an artifact is completed.

## Verification

362 focused and integration tests pass. The complete 2831-test suite did not finish within the available window; its first 20 failures were compared with unchanged v225.2. Two roadmap-contract failures were fixed, and the remaining 18 in that sample reproduce in the baseline.
