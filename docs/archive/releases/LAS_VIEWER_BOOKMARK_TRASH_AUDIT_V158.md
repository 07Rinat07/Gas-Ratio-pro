# LAS Viewer Bookmark Trash Audit v158

Добавлен renderer-neutral журнал операций корзины закладок.

Журнал фиксирует:
- удаление закладки в корзину;
- восстановление;
- окончательное удаление;
- автоматическую очистку по retention policy.

История сохраняется в recent-session preferences, ограничена 200 событиями и может быть очищена независимо от содержимого корзины.
