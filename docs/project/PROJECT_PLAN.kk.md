# Gas Ratio Pro жоба жоспары

Жаңартылған күні: 18 шілде 2026 жыл. Белсенді жинақ: `v225.3`.

## Міндетті инженерлік қағидалар

- басып шығару үшін инженерлік деректер, тректер және шкалалар қысқартылмайды;
- UI, PDF, DOCX, SVG және PNG renderer-neutral contracts пайдаланады;
- бір pipeline page geometry үшін жалғыз дереккөз болады;
- A4/A3 физикалық параметрлерді және ең төмен оқылатын типографиканы қолданады;
- экспорт бір әрекетпен іске қосылады және пакет аяқталғанша қайталама іске қосуды бұғаттайды;
- пайдаланушы және developer нұсқаулықтары орыс, қазақ және ағылшын тілдерінде синхронды жаңартылады.

## Аяқталған кезең — v225.3

- ортақ page-space колонтитулдары және бет нөмірлеуі;
- renderer-neutral label/legend model негізіндегі қайталанатын шартты белгілер;
- geometry signature v3;
- page-aware package v1.1;
- жергіліктендірілген Print Center summary;
- LAS Viewer SVG/PDF/PNG бір пакет арқылы;
- QA render-model және page-chrome primitives ескереді.

## Келесі кезең — Visible Print Center Integration

1. Экспортқа дейін Professional Print Center ішінде физикалық профиль мен нақты бет санын көрсету.
2. Бар page-aware output contract-ты PDF, DOCX, SVG және PNG форматына беру.
3. `page_svgs` мәндерін DOCX/HTML қабатына layout-ты қайта құрмай қосу.
4. Бүкіл пакет орындалғанша қайталама іске қосуды бұғаттау.
5. Parity тексерілгеннен кейін тәуелсіз Plotly/static fallback жолдарын жою.
6. Кодты, тесттерді, README, нұсқаулықтарды, күйді, changelog және жоспарларды `ru / kk / en` тілдерінде жаңарту.
