# Ағымдағы күй — v225.3

Жаңартылған күні: 18 шілде 2026 жыл.

**Page Chrome & Print Center Contract** кезеңі аяқталды:

- `VisualizationPrintLayout` 2.1 нұсқасына жаңартылды;
- физикалық бетке бөлек `header_bounds`, `footer_bounds`, `legend_bounds` және `chrome_primitives` қосылды;
- тақырып, қосымша тақырып, жіктеу, құжат коды, footer, бет нөмірі және қайталанатын шартты белгілер `page_pt` координаттарында бір рет құрылады;
- орыс, қазақ және ағылшын тілдеріндегі бет нөмірлеуі мен шартты белгілер тақырыптары синхрондалды;
- SVG және PDF бірдей page-space primitives пайдаланады, PNG сол SVG беттерінен жасалады;
- geometry signature v3 физикалық бет аймақтары мен page chrome-ды қамтиды;
- `VisualizationPageAwarePackage` 1.1 нұсқасына жаңартылды;
- нақты профильді, бағдарды, DPI және бет санын көрсететін `VisualizationPrintCenterService` қосылды;
- LAS Viewer экспорты SVG/PDF/PNG үшін бір page-aware пакетке көшірілді;
- single-page fallback өшірулі күйде қалды.
- фондық экспорт `ThreadPoolExecutor` арқылы іске қосылады, сондықтан progress пен cooperative cancellation орындалу кезінде жұмыс істейді;

Пайдаланушы және әзірлеуші құжаттамасы `ru / kk / en` тілдерінде синхронды жаңартылды, manifest толықтырылды.

Келесі рұқсат етілген кезең: физикалық пакет жиынтығын көрінетін Professional Print Center интерфейсіне тікелей қосу, DOCX/HTML үшін page-aware preview беру және parity тексерілгеннен кейін тәуелсіз legacy static-export жолдарын жою.

## Шығарылымды басқару

Ағымдағы кезең: **Stabilization & Release Audit**. **Release candidate v225.3** бірыңғай page-aware pipeline, үш тілдегі құжаттама және қайталанатын эталондық артефактілер бойынша тексеріледі.

## v225.3 тексеруі

- 362 профильдік, renderer, Print Center, background-export және documentation governance тесті өтеді;
- Python compileall және салыстырмалы сілтемелерді тексеру өтеді;
- 2831 тесттен тұратын толық suite қолжетімді орындалу терезесінде аяқталмады; алғашқы 20 қате диагностикалық үлгі ретінде бастапқы v225.2 архивімен салыстырылды;
- roadmap-contract бойынша 2 қате v225.3 ішінде түзетілді, ал осы үлгідегі қалған 18 қате v225.2-де өзгеріссіз қайталанады; кейінгі ықтимал legacy қателері жіктелмеген;
- ескірген legacy-UI тест келісімшарттарының толық бөлек аудиті аяқталғанша жинақ release candidate күйінде қалады.
