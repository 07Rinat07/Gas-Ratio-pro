# Gas Ratio Pro — Professional Export Wizard increment

Дата: 2026-07-13

## Реализовано

1. Добавлен renderer-neutral модуль `reports/export_wizard.py`.
2. Реализована пятиэтапная модель мастера экспорта:
   - источник;
   - состав отчета;
   - формат;
   - каталог назначения;
   - проверка перед экспортом.
3. Добавлены безопасные переходы между этапами с ограничением диапазона.
4. Добавлен side-effect-free preflight, который можно вызывать после каждого изменения UI.
5. Добавлена матрица возможностей профессионального отчета:
   - PDF;
   - DOCX;
   - PDF + DOCX bundle.
6. Исправлен опасный fall-through в `build_ui_export_artifact`:
   - ранее PNG/SVG/XLSX могли возвращать ZIP с PDF/DOCX и неверным MIME;
   - теперь для неподдерживаемого renderer-маршрута возникает явная диагностическая ошибка.
7. Добавлены тесты навигации, preflight, обязательного источника и блокировки неподдерживаемых форматов.

## Архитектурные решения

- Мастер не зависит от Streamlit.
- Мастер не пересчитывает инженерные данные.
- Мастер создает существующий `PresentationExportUiState` и передает его текущему export layer.
- Форматы PNG/SVG/XLSX остаются в общем UI-каталоге, но должны направляться в специализированные visualization/table exporters.
