# Руководство пользователя

## Назначение

Gas Ratio Interpreter v0.3 помогает импортировать газовые данные, привести
колонки к стандартным полям, рассчитать газовые коэффициенты, построить
палетки и получить предварительную инженерную интерпретацию.

Программа не заменяет геологическую экспертизу. Все выводы нужно проверять
по ГИС, литологии, буровому контексту, фону, СПО, наращиваниям и рециркуляции.

## Рабочий сценарий

1. Запустите приложение:

```powershell
streamlit run app/streamlit_app.py
```

2. Загрузите файл `CSV`, `XLSX` или `XLSM`.
3. Для Excel выберите нужный лист.
4. Проверьте первые 20 строк в превью.
5. Проверьте строку заголовков.
6. Проверьте сопоставление колонок.
7. Просмотрите предупреждения.
8. Выберите режим `Ch`.
9. Выберите интервал.
10. Проверьте Pixler, ternary и depth-графики.
11. При необходимости скачайте расчетную таблицу через `Экспорт CSV`.

## Строка заголовков

Приложение ищет строку заголовков среди первых 50 строк. Если файл содержит
служебные строки сверху, например название отчета или дату выгрузки, программа
должна найти настоящую строку с `Depth`, `C1`, `C2` и другими колонками.

Если автоопределение ошиблось, укажите номер строки вручную. Нумерация в поле
интерфейса начинается с `0`.

## Mapping колонок

После выбора строки заголовков приложение пытается сопоставить входные колонки
со стандартными полями:

```text
well, depth, depth_from, depth_to,
c1, c2, c3, ic4, nc4, ic5, nc5,
co2, h2s, rop, lithology
```

Если колонка не распознана автоматически, выберите ее вручную в блоке mapping.

## Предупреждения

Предупреждения не всегда означают ошибку. Они показывают, где данные неполные
или требуют внимания:

- отсутствует компонент C1-C5;
- отсутствует глубина;
- часть значений не удалось преобразовать в число;
- формула `Ch` требует подтверждения;
- выбран резервный режим `Ch`.

Если компонент C1-C5 отсутствует, он принимается равным `0`, чтобы расчет мог
продолжиться. Это обязательно нужно учитывать при интерпретации.

## Выбор интервала

Интервалы отображаются по индексу строки и глубине. Если глубина отсутствует
или полностью нечисловая, приложение использует индекс строки как техническую
глубину и показывает предупреждение.

## Экспорт

В v0.3 доступен экспорт расчетной таблицы в CSV из интерфейса. Файл содержит
исходные сопоставленные поля, расчетные коэффициенты, классификацию и текстовое
предупреждение по интерпретации.
