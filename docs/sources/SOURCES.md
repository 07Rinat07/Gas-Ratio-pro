# Источники проекта GAS RATIO PRO

В этой папке хранятся PDF-источники, которые используются как доказательная база для инженерных решений, спецификаций и алгоритмов проекта.

## Правило хранения источников

В документации проекта нельзя ссылаться только на локальные пути пользователя вида `C:\Users\...`. Каждый внешний источник должен быть сохранен внутри проекта и зарегистрирован в `source_registry.json`.

Правильный формат ссылки:

```text
docs/sources/<имя-файла>.pdf
```

## Зарегистрированные источники

1. `gladkov-geological-hydrodynamic-modeling.pdf` — учебное пособие Е.А. Гладкова по геологическому и гидродинамическому моделированию месторождений нефти и газа.
2. `lab-4-property-cubes.pdf` — методический материал по расчету кубов свойств: литология, песчанистость, пористость, проницаемость, флюидные контакты, геометрические свойства и нефтенасыщенность.
3. `mud-gas-analysis-reservoir-evaluation.pdf` — статья по применению анализа бурового газа для оценки коллектора.

## Назначение

Папка `docs/sources/` нужна для того, чтобы при проверке проекта можно было открыть сам первоисточник, а не искать файл на локальном компьютере автора.
