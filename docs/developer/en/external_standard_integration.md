# Integrating an external standard or library

1. Record the format, supported revisions and acceptance criteria.
2. Locate the official specification and test data with clear rights.
3. Review the library license and register the component decision.
4. Implement an adapter without exposing library objects to the domain core.
5. Register the format and implement a metadata-only scanner.
6. Preserve the source as an immutable artifact and create Dataset Manifest/provenance records.
7. Add bounded-memory, security, QC and conformance tests.
8. Add ru/kk/en guidance and update the documentation manifest.
