# Формат плагиндерінің тізілімі

`FormatPluginRegistry` қолданыстағы `DataFormatCapability` келісімін metadata scanner, quick QC және importer/exporter идентификаторларымен байланыстырады. Үшінші тарап кітапханаларының ауыр нысандары адаптер шекарасынан шықпайды.

Capability matrix — Workbench үшін JSON-қа қауіпсіз келісім. Жаңа плагин тіркелген форматты қолдануы, тесттерге, лицензиялық жазбаға және үш тілдегі құжаттамаға ие болуы тиіс.
