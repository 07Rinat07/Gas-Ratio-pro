# Сыртқы стандартты немесе кітапхананы интеграциялау

1. Форматты, нұсқаларды және қабылдау өлшемдерін бекіту.
2. Ресми спецификацияны және құқықтары анық сынақ деректерін табу.
3. Кітапхана лицензиясын тексеріп, компоненттер тізілімін толтыру.
4. Кітапхана нысандарын домендік ядроға өткізбейтін адаптер жасау.
5. Форматты Format Registry-де тіркеп, metadata-only scanner жасау.
6. Бастапқы файлды immutable artifact ретінде сақтап, Dataset Manifest/provenance құру.
7. Bounded-memory, security, QC және conformance сынақтарын қосу.
8. ru/kk/en нұсқаулықтарын қосып, documentation manifest-ті жаңарту.
