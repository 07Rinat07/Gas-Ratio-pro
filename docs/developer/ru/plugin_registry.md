# Реестр плагинов форматов

`FormatPluginRegistry` связывает существующий `DataFormatCapability` с metadata scanner, quick QC и идентификаторами importer/exporter. Тяжёлые объекты сторонних библиотек не выходят за границу адаптера.

Capability matrix является JSON-безопасным контрактом Workbench. Новый плагин обязан использовать уже зарегистрированный формат, иметь тесты, лицензионную запись и документацию на трёх языках.
