# Интеграция внешнего стандарта или библиотеки

1. Зафиксировать формат, версии и критерии приёмки.
2. Найти официальную спецификацию и тестовые данные с понятными правами.
3. Проверить лицензию библиотеки и заполнить реестр компонентов.
4. Создать адаптер, не передавая библиотечные объекты в доменное ядро.
5. Зарегистрировать формат в Format Registry и реализовать metadata-only scanner.
6. Сохранять исходник как immutable artifact и создавать Dataset Manifest/provenance.
7. Добавить bounded-memory, security, QC и conformance-тесты.
8. Добавить инструкции ru/kk/en и обновить documentation manifest.
