# Supported formats and lawful use of external material

GAS RATIO PRO is developed around open industry standards. The product supports or plans support for LAS 1.x/2.x/3.x, DLIS, LIS79, SEG-Y, RESQML, GRDECL, GeoPackage, Shapefile, GeoTIFF, HDF5 and NetCDF.

Source files are retained without hidden modification. Archival LAS files earlier than 2.0 are opened through an explicit compatibility mode. Before conversion, the application presents warnings and stable validation codes.

Documentation, examples and libraries are used only when the source is lawful and the license is compatible. Closed commercial source code is not copied. Commercial-system integration uses open exchange formats, official APIs or licensed SDKs.
