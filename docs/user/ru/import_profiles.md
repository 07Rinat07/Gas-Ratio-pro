# Профили импорта

Профиль импорта сохраняет безопасные настройки предварительного анализа и импорта для конкретного формата. Профили принадлежат проекту и не изменяют исходные файлы.

## Основные возможности

- выбор формата и версии scanner;
- сохранение strict/tolerant режима LAS;
- настройки DLIS logical file/frame/channel;
- настройки байтов SEG-Y;
- повторное использование при пакетном импорте;
- readiness score перед дальнейшей обработкой.

Кэш предпросмотра использует SHA-256 файла, идентификатор профиля и версию scanner. Изменение любого компонента запускает новое сканирование.
