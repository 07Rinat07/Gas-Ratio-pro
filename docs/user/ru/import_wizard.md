# Мастер импорта

Мастер импорта проводит файл через этапы выбора, предпросмотра, настройки, быстрой проверки качества и регистрации Dataset.

## Пакетный импорт

Можно обработать несколько файлов одной операцией. Ошибка одного файла не останавливает остальные. Для каждого элемента сохраняется отдельный статус, код ошибки и идентификатор зарегистрированного Dataset.

## Готовность данных

После metadata preview и quick QC рассчитывается показатель готовности от 0 до 100. Статусы: `ready`, `review`, `blocked`. Показатель является рекомендацией и не заменяет инженерную проверку.

## Безопасность

Исходные файлы сохраняются неизменяемыми. Состояние мастера не содержит содержимого файлов, DataFrame или объектов сторонних библиотек.
