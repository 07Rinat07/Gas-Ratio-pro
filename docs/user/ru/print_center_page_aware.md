# Центр печати, физические страницы и контроль visual baseline

Revision: 5 · GAS RATIO PRO v225.7

## Основной пользовательский процесс

1. Выберите A4/A3, ориентацию, язык и встроенный либо пользовательский физический профиль.
2. Нажмите **«Рассчитать точный физический пакет»**.
3. Проверьте число страниц, каждую SVG-страницу и статус cross-format parity.
4. Запустите PDF, DOCX, HTML, SVG/PNG или комбинированный экспорт.
5. При ошибке parity не заменяйте эталоны вручную: сохраните диагностику и повторите расчёт после исправления исходных параметров.

## Единый пакет

PDF, SVG, PNG, DOCX и HTML используют один page-aware пакет. DOCX/HTML получают канонический массив `pages`; повторная пагинация и fallback на первую страницу запрещены. Многостраничные SVG/PNG выдаются ZIP-пакетом с `manifest.json`.

## Читаемость

В v225.7 PDF и DOCX используют единый print-readability contract. Минимальные размеры шрифта легенды, raster dimensions и схема «одна запись легенды на строку» проверяются автоматически. Пользовательский профиль не может снизить безопасные ограничения A4/A3.

## Visual baseline

Четыре физические golden-профиля A4/A3 продолжают проверяться по SVG/PNG/PDF. Дополнительно 13 исторических визуальных тестов переведены на semantic rebaseline: проверяется назначение элементов — диапазон глубины, интервалы, легенда, заголовки, priority frame, ternary regions и fill modes — вместо случайного числа внутренних trace.

## Языки

Интерфейс, сообщения preview, инструкции и диагностика поддерживаются на русском, казахском и английском языках.
