# Адаптивный макет инженерных отчётов

Revision 1 · Gas Ratio Pro v225.9

## Что исправлено

Ориентация страницы и размер содержимого теперь связаны одним физическим контрактом. Для A3 landscape графики, подписи, таблицы и текстовые разделы используют фактическую полезную область листа, а не прежнюю фиксированную ширину, рассчитанную под меньший формат.

## Поведение по умолчанию

- выбранная ориентация A4/A3 применяется к реальной странице PDF и секции DOCX;
- график масштабируется по доступным ширине и высоте страницы с сохранением пропорций;
- метаданные, легенды, статистика и текстовые таблицы занимают полную рабочую ширину;
- HTML-графики растягиваются по ширине контейнера и адаптируются к окну просмотра;
- минимальные размеры шрифта, линий и треков по-прежнему защищены физическим профилем;
- page-aware preview и финальный экспорт используют одинаковую геометрию.

## Практический результат для A3 landscape

В альбомном A3 инженерный планшет больше не располагается узкой колонкой слева. Он занимает почти всю полезную ширину и высоту листа. Текстовые отчёты и широкие таблицы также растягиваются по рабочему фрейму, поэтому справа не остаётся неоправданно большой пустой области.

## Поддерживаемые форматы

- **PDF:** размеры вычисляются из фактического ReportLab frame;
- **DOCX:** ширина определяется параметрами текущей section;
- **HTML:** используется responsive-контейнер шириной 100%;
- **SVG/PNG preview:** сохраняется page-aware физическая геометрия.

## Проверка

Контракт `print-readability/v1.1` требует политики `available-frame`, page-aware aspect ratio и не менее 90% допустимого использования landscape-фрейма для широкого содержимого. Изменение этого поведения допускается только через контролируемый visual rebaseline.
