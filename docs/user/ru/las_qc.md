# Контроль качества LAS

**Ревизия:** 1

Модуль LAS QC проверяет глубину, пропуски, диапазоны, скачки, постоянные участки и единицы измерения. Проверка не изменяет исходный LAS-файл.

## Результаты

- `passed` — критических отклонений не обнаружено;
- `warning` — найдены предупреждения;
- `failed` — найдены ошибки, требующие проверки инженером.

Каждое замечание имеет стабильный код `QC-*`, который одинаков на русском, казахском и английском языках.
