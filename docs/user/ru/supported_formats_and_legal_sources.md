# Поддерживаемые форматы и законное использование внешних материалов

GAS RATIO PRO развивается на основе открытых отраслевых стандартов. Проект поддерживает или планирует поддержку LAS 1.x/2.x/3.x, DLIS, LIS79, SEG-Y, RESQML, GRDECL, GeoPackage, Shapefile, GeoTIFF, HDF5 и NetCDF.

Исходные файлы сохраняются без скрытого изменения. Архивные LAS старее 2.0 открываются в явном режиме совместимости. Перед преобразованием приложение показывает предупреждения и стабильные коды проверки.

Документация, примеры и библиотеки используются только при наличии законного источника и совместимой лицензии. Закрытый код коммерческих продуктов не копируется. Интеграция с коммерческими системами выполняется через открытые форматы, официальные API или лицензированные SDK.
