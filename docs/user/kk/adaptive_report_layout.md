# Инженерлік есептердің бейімделетін макеті

Revision 1 · Gas Ratio Pro v225.9

## Не түзетілді

Бет бағыты мен мазмұн өлшемі енді бір физикалық контракт арқылы басқарылады. A3 landscape режимінде графиктер, жазулар, кестелер және мәтіндік бөлімдер кіші форматқа арналған бұрынғы тұрақты енді емес, парақтың нақты пайдалы аймағын пайдаланады.

## Әдепкі әрекет

- таңдалған A4/A3 бағыты PDF бетіне және DOCX section-ға нақты қолданылады;
- график пропорциясын сақтап, қолжетімді ен мен биіктікке масштабталады;
- metadata, legend, statistics және мәтіндік кестелер жұмыс аймағының толық енін пайдаланады;
- HTML графиктері контейнердің 100% еніне бейімделеді;
- қаріп, сызық және track минимал өлшемдері физикалық профильмен қорғалады;
- page-aware preview және соңғы export бір геометрияны пайдаланады.

## A3 landscape нәтижесі

Альбомдық A3 форматында инженерлік планшет енді сол жақтағы тар баған болып қалмайды. Ол парақтың пайдалы ені мен биіктігінің басым бөлігін алады. Мәтіндік есептер мен кең кестелер де толық frame бойынша созылады, сондықтан оң жақта негізсіз үлкен бос аймақ қалмайды.

## Қолдау көрсетілетін форматтар

- **PDF:** өлшемдер нақты ReportLab frame бойынша есептеледі;
- **DOCX:** ені ағымдағы section параметрлерінен алынады;
- **HTML:** 100% енді responsive контейнер қолданылады;
- **SVG/PNG preview:** page-aware физикалық геометрия сақталады.

## Тексеру

`print-readability/v1.1` контракты `available-frame` саясатын, page-aware aspect ratio және кең мазмұн үшін landscape frame-нің кемінде 90% пайдаланылуын талап етеді. Бұл baseline тек controlled visual rebaseline арқылы өзгертіледі.
