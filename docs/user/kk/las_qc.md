# LAS сапасын бақылау

**Ревизия:** 1

LAS QC модулі тереңдікті, бос мәндерді, диапазондарды, секірістерді, тұрақты аралықтарды және өлшем бірліктерін тексереді. Тексеру бастапқы LAS файлын өзгертпейді.

## Нәтижелер

- `passed` — маңызды ауытқулар табылған жоқ;
- `warning` — ескертулер табылды;
- `failed` — инженер тексеруі тиіс қателер табылды.

Әр ескертуде орыс, қазақ және ағылшын тілдерінде бірдей тұрақты `QC-*` коды бар.
