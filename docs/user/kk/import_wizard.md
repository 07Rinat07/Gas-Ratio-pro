# Импорт шебері

Импорт шебері файлды таңдау, алдын ала қарау, баптау, жедел сапа тексеруі және Dataset тіркеу кезеңдері арқылы өткізеді.

## Пакеттік импорт

Бір операцияда бірнеше файлды өңдеуге болады. Бір файлдың қатесі қалғандарын тоқтатпайды. Әр файл үшін жеке күй, қате коды және тіркелген Dataset идентификаторы сақталады.

## Деректердің дайындығы

Metadata preview және quick QC нәтижелері бойынша 0-ден 100-ге дейін дайындық көрсеткіші есептеледі. Күйлер: `ready`, `review`, `blocked`. Бұл көрсеткіш инженерлік тексеруді алмастырмайды.

## Қауіпсіздік

Бастапқы файлдар өзгертілмей сақталады. Шебер күйінде файл мазмұны, DataFrame немесе сыртқы кітапхана нысандары болмайды.

## Пакеттік импорт және фондық тапсырмалар

Data Workspace бөлімінде «Кәсіби импорт шеберін» ашып, бірнеше файлды таңдаңыз және «Импортты бастау» батырмасын басыңыз. Әр файл тәуелсіз өңделеді. Тапсырмалар кестесінде күй, орындалу пайызы, сәтті және қате элементтер саны көрсетіледі. Аяқталған тапсырмада тек сәтсіз файлдарды қайталауға болады. Терминалдық тарих жоба ішінде `imports/history.jsonl` файлына сақталады.


## Қалпына келтіру, бас тарту және тарих

Қайта іске қосылғаннан кейін аяқталмаған тапсырма `interrupted` күйін алады және автоматты түрде қайта орындалмайды. Пайдаланушы оны анық түрде қайталай алады. Белсенді тапсырмадан бас тартуға болады. Тарихты сүзуге, іздеуге және JSON немесе CSV түрінде жүктеуге болады. Staging тазалау белсенді тапсырмалар қолданбайтын файлдарды ғана жояды.
