# Қолдау көрсетілетін форматтар және сыртқы материалдарды заңды пайдалану

GAS RATIO PRO ашық салалық стандарттар негізінде дамытылады. Жоба LAS 1.x/2.x/3.x, DLIS, LIS79, SEG-Y, RESQML, GRDECL, GeoPackage, Shapefile, GeoTIFF, HDF5 және NetCDF форматтарын қолдайды немесе қолдауды жоспарлайды.

Бастапқы файлдар жасырын өзгертілмей сақталады. LAS 2.0-ден ескі мұрағаттық файлдар айқын үйлесімділік режимінде ашылады. Түрлендіру алдында қолданба ескертулер мен тұрақты тексеру кодтарын көрсетеді.

Құжаттама, мысалдар және кітапханалар тек заңды дереккөз және үйлесімді лицензия болған кезде пайдаланылады. Коммерциялық өнімдердің жабық коды көшірілмейді. Интеграция ашық форматтар, ресми API немесе лицензияланған SDK арқылы орындалады.
