# Импорт профильдері

Импорт профилі нақты формат үшін алдын ала талдау мен импорттың қауіпсіз баптауларын сақтайды. Профильдер жобаға тиесілі және бастапқы файлдарды өзгертпейді.

## Негізгі мүмкіндіктер

- формат пен scanner нұсқасын таңдау;
- LAS strict/tolerant режимін сақтау;
- DLIS logical file/frame/channel баптаулары;
- SEG-Y байттарының баптаулары;
- пакеттік импортта қайта пайдалану;
- кейінгі өңдеуге дейін readiness score.

Алдын ала қарау кэші файлдың SHA-256 мәнін, профиль идентификаторын және scanner нұсқасын пайдаланады. Кез келген бөлік өзгерсе, сканерлеу қайта орындалады.
