# Басып шығару орталығы және тікелей көпбетті preview

Revision: 3 · GAS RATIO PRO v225.5

## Не өзгерді

Professional Print Center енді физикалық page-aware пакетке тікелей қосылған. Экспорт басталғанға дейін интерфейс мыналарды есептеп көрсетеді:

- A4/A3 пішімі және бағдар;
- DPI және беттердің нақты саны;
- пакеттің дайындық күйі;
- әр физикалық SVG бетінің жеке preview-ы;
- PDF, SVG, PNG, DOCX және HTML үшін бірдей трек бөлінісі.

## Жұмыс тәртібі

1. Есеп бөлімдерін, қағаз пішімін, бағдарды және құжат тілін таңдаңыз.
2. **«Нақты физикалық пакетті есептеу»** әрекетін орындаңыз.
3. Профильді, бет санын және preview тізіміндегі әр бетті тексеріңіз.
4. PDF, DOCX, HTML немесе біріктірілген PDF+DOCX экспортын іске қосыңыз.
5. Ағымдағы пакет аяқталғанша екінші экспортты бастамаңыз.

## DOCX және HTML

DOCX пен HTML 1.1 нұсқасындағы preview келісімшартының канондық `pages` массивін алады. Әр жазбада бет нөмірі, физикалық өлшем, тректер тізімі және дайын SVG бар. Бұл форматтар layout-ты қайта есептемейді және бірінші бет өрісін fallback ретінде пайдаланбайды.

## Тілдер

Тректер, қисықтар, аралықтар және беттер жиынтығы, бет белгілері, қате хабарлары және пакет күйі таңдалған орыс, қазақ немесе ағылшын тілінде жасалады.

## Шектеулер

Бір беттік fallback өшірілген. Канондық беттер массиві жоқ болса немесе жарияланған `page_count` мәніне сәйкес келмесе, пакет қате деп есептеледі және тек бірінші бет ретінде үнсіз экспортталмайды.

## Cross-format parity және пайдаланушы профильдері

Файл берілмес бұрын пакет автоматты parity gate тексеруінен өтеді. Ол SVG, PNG, PDF және тікелей DOCX/HTML preview үшін бет санын, физикалық өлшемдерді, трек бөлінісін және geometry signature мәнін салыстырады. Айырмашылық болса, экспорт бұғатталады.

Бет параметрлерінде кірістірілген профильді таңдауға немесе A4/A3 пайдаланушы профилін сақтауға болады. Профиль пішімді, бағдарды, жиектерді, DPI, ең кіші қаріпті, трек енін және беттегі трек санын сақтайды. Оқылым мәндері қауіпсіз базалық профильден төмен бола алмайды.

SVG немесе PNG бірнеше беттен тұрса, қолданба жеке бет файлдары және `manifest.json` бар ZIP-пакет жасайды. Бірінші бет енді толық құжат ретінде берілмейді.
