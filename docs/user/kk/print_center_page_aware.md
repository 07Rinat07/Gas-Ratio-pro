# Басып шығару орталығы, физикалық беттер және visual baseline бақылауы

Revision: 6 · GAS RATIO PRO v225.8 stable

## Негізгі пайдаланушы үрдісі

1. A4/A3, бағдар, құжат тілі және кірістірілген немесе пайдаланушы профилін таңдаңыз.
2. **«Нақты физикалық пакетті есептеу»** батырмасын басыңыз.
3. Беттер санын, әр SVG-бетті және cross-format parity күйін тексеріңіз.
4. PDF, DOCX, HTML, SVG/PNG немесе біріктірілген экспортты іске қосыңыз.
5. Parity қатесі болса, эталонды қолмен алмастырмаңыз: диагностиканы сақтап, бастапқы параметрлерді түзеткеннен кейін қайта есептеңіз.

## Бір пакет

PDF, SVG, PNG, DOCX және HTML бір page-aware пакетті пайдаланады. DOCX/HTML канондық `pages` массивін алады; қайта пагинация және бірінші бетке fallback тыйым салынған. Көпбетті SVG/PNG `manifest.json` бар ZIP ретінде беріледі.

## Оқылымдылық

v225.7 нұсқасында PDF және DOCX ортақ print-readability contract қолданады. Легенда қарпінің ең аз өлшемі, raster dimensions және «бір жолда бір белгі» схемасы автоматты түрде тексеріледі. Пайдаланушы профилі A4/A3 қауіпсіз шектерін төмендете алмайды.

## Visual baseline

A4/A3 төрт physical golden профилі SVG/PNG/PDF бойынша тексеріледі. Сонымен бірге 13 тарихи визуалдық тест semantic rebaseline-ға көшірілді: ішкі trace санының орнына тереңдік диапазоны, интервал, легенда, тақырып, priority frame, ternary regions және fill modes мағынасы тексеріледі.

## Тілдер

Интерфейс, preview хабарлары, нұсқаулықтар және диагностика орыс, қазақ және ағылшын тілдерінде қолжетімді.
