# Roadmap Documentation

This directory stores project roadmap documents, phase summaries, redesign notes,
and implementation continuation files.

Repository root must stay clean. Only operational root files are allowed there,
for example `README.md`, `CHANGELOG.md`, `LICENSE`, dependency files, launch
scripts, and configuration files.

Roadmap documents must be placed here instead of the repository root:

- `ROADMAP_v*.md`
- `GAS_RATIO_PRO_ROADMAP_*.md`
- `GAS_RATIO_PRO_ROADMAP_*.txt`
- phase summaries
- implementation continuation guides
- workspace implementation notes
