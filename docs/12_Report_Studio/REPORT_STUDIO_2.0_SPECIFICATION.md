# GAS RATIO PRO — REPORT STUDIO 2.0 SPECIFICATION

## 1. Цель

Report Studio 2.0 должен заменить неудобную зависимость от HTML и дать пользователю профессиональные форматы вывода.

---

## 2. Форматы

Обязательные:

- PDF;
- DOCX;
- XLSX;
- PNG;
- SVG;
- TIFF/JPEG.

Дополнительные:

- HTML;
- Markdown;
- ZIP package.

---

## 3. Well Interpretation Report

Отчет должен включать:

- титульную информацию по скважине;
- QC summary;
- mud gas interpretation;
- petrophysical results;
- fluid intervals;
- pay zones;
- plots;
- reservoir volumes;
- conclusion.

---

## 4. Excel Workbook

Листы:

- Summary;
- Curves;
- QC;
- Mud Gas;
- Petrophysics;
- Intervals;
- Fluids;
- Volumetrics;
- Sources.

---

## 5. Print Interval

Для любого отчета пользователь может задать:

- Depth From;
- Depth To;
- selected zones;
- selected tracks;
- page size;
- orientation.
