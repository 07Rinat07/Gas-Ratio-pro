# Document Model v29

`Document Model` вводит общий объект документа для всех форматов представления Gas Ratio Pro.

## Назначение

Инженерская информация собирается один раз из `PresentationModel`, после чего HTML, будущие PDF/DOCX и UI-рендереры только отображают готовые блоки документа.

## Правило

Не дублировать инженерскую интерпретацию между форматами представления.

```text
PresentationModel
        ↓
EngineeringDocument
        ↓
HTML / PDF / DOCX / UI
```

## Основные блоки

- `DocumentMetadata` — заголовок, подзаголовок, профиль, служебные строки шапки.
- `DocumentSection` — логический раздел документа.
- `DocumentTable` — таблица отчета.
- `DocumentPlot` — планшет или графический блок.
- `DocumentNotice` — информационное примечание.
