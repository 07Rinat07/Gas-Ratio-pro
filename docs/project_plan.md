# План проекта Gas Ratio Interpreter

## Цель

Gas Ratio Interpreter развивается как локальное инженерное приложение для работы
с газовым каротажем, LAS-файлами, корреляцией скважин, расчетами, графиками,
экспортами и воспроизводимой документацией.

Приложение должно быть понятным для инженера: загрузить данные, проверить
mapping, увидеть предупреждения, построить графики, сохранить скважины и получить
печатный результат без дополнительных внешних инструкций.

## Текущий статус

Готово:

- импорт `LAS`, `CSV`, `XLSX`, `XLSM`;
- рабочий workflow с мультизагрузкой файлов;
- LAS-редактор: проверка глубины, исправление убывающего порядка, смена шага,
  добавление строк, ручная правка и экспорт;
- локальное хранение скважин и версий данных;
- расчет `Wh`, `Bh`, `BAR2`, Pixler ratios, ternary ratios и режима `Ch`;
- предварительная интерпретация интервалов по явным правилам;
- Pixler, ternary и depth-графики;
- вкладка `LAS-корреляция` для нескольких LAS: ГИС рядом с газами;
- HTML-выгрузка графиков для печати;
- локальное логирование в `logs/app.log`;
- документация по запуску, данным, формулам, LAS-редактору, корреляции и ошибкам;
- pytest и preflight для проверки проекта.

## Ближайшие этапы

### Этап 1. Справка и подсказки

- Добавить больше подсказок по выбранному интервалу.
- Сделать справочные панели по формулам `Wh/Bh/Ch/BAR2`.
- Добавить понятные причины `NaN` прямо в интерфейс.
- Добавить мастер диагностики mapping: какие колонки не найдены и что выбрать.
- Добавить ссылки из предупреждений на разделы документации.

Критерий готовности: пользователь понимает причину предупреждения прямо в
workflow и может проверить ее по документации.

### Этап 2. LAS-редактор

- Расширить автопроверку пропусков глубины.
- Добавить ручное добавление строк по интервалу.
- Добавить массовые операции: удалить дубли глубин, обрезать интервал, заменить
  NULL, проверить шаг.
- Сделать предпросмотр до/после правки LAS.

### Этап 3. LAS-корреляция

- Добавить ручной override групп кривых.
- Добавить индивидуальный X-scale по трекам.
- Добавить режим сравнения одной кривой между скважинами.
- Добавить таблицу выбранного интервала под графиком.
- Улучшить HTML-печать планшетов.

### Этап 4. Проекты и скважины

- Создать карточку проекта.
- Хранить исходные LAS, подготовленные LAS, mapping, настройки графиков и версии
  экспортов.
- Добавить загрузку сохраненного проекта.
- Добавить экспорт выбранных скважин в `LAS`, `XLSX`, `CSV`, `HTML`.

### Этап 5. Интерпретационные графики и отчеты

- Добавить графики и маркеры по литературному источнику mud gas analysis.
- Добавить зоны oil/gas/dry gas/non-productive/contact как редактируемые слои.
- Добавить печатный отчет по выбранному интервалу.
- Подготовить экспорт `PDF/PNG/SVG`.

### Этап 6. Формулы и методика

- Сверить `Wh/Bh/Ch/BAR2/Pixler` с источниками и корпоративной методикой.
- Добавить inverse oil indicator.
- Добавить warnings по нулям, пропускам, единицам измерения и плохому mapping.
- Все новые формулы покрывать тестами и документировать.

## Правило развития

Новая помощь пользователю добавляется как проверяемая логика: правило, таблица,
график, предупреждение, мастер, отчет или документация. Каждое изменение должно
быть воспроизводимым и покрываться тестом там, где это возможно.
