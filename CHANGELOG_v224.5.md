# Gas Ratio Pro v224.5

## Report localization

- The selected document language is now passed into the report model.
- PDF/DOCX report metadata, section titles, plot titles, fluid labels and common engineering table headers are localized for RU, KK and EN.
- Kazakh and English reports no longer silently use Russian report chrome and core section names.
- Added report-local translation helpers and regression tests.
- Technical mnemonics (TGAS, C1-C5, Wh, Bh, Ch, Oil Index) remain unchanged.
